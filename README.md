## 🌟 About This Project

<img align="right" src="https://media.giphy.com/media/qgQUggAC3Pfv687qPC/giphy.gif" width="280" alt="coding gif"/>

> 🎓 **CSE-Merto** is a one-stop learning hub for **Metropolitan University CSE students**, packed with all the **assignments, lab reports, codes, notes, and resources** you'll ever need.

✨ Whether you're a **junior** struggling with your first C program, or a **senior** revising for your final viva — this repo has something for everyone.

🎯 **Mission:** Make learning **easier**, **collaborative**, and **fun** for every CSE student walking the corridors of Metropolitan University.

📦 Curated by students, for students — covering **multiple batches** (CSE-56 → CSE-62) and **all major subjects**.

<br clear="right"/>

### 💌 A Heartfelt Message from the Author

</div>

> 🌍 **Hey fellow learner!** 👋
>
> This isn't just a repo — it's a **growing community** of passionate Metropolitan CSE students helping each other succeed. Whether you've solved a tricky problem, taken great notes in class, or written a clean lab report — **your contribution matters**. 💙
>
> 📚 **Got something valuable?** A neat algorithm, a corrected solution, a beautiful note? Don't keep it to yourself — **share it with the world** and help juniors who'll walk the same path tomorrow.
>
> 🚀 **Together, we make learning easier.** No effort is too small. Every commit, every PR, every star ⭐ helps someone, somewhere.
>
> 💬 **Join us. Contribute. Collaborate. Grow.** Let's build the **biggest CSE knowledge vault** Metropolitan has ever seen!

### 🎯 How to Contribute

```bash
# 1. Fork the repo (top-right corner of GitHub) 🍴
# 2. Clone YOUR fork
git clone https://github.com/<your-username>/CSE-Merto.git

# 3. Create a new branch
git checkout -b feature/awesome-notes

# 4. Add your magic ✨
git add .
git commit -m "✨ Added Sem 2.1 DS sorting notes"

# 5. Push it
git push origin feature/awesome-notes

# 6. Open a Pull Request — done! 🎉
```

### 📋 Contribution Guidelines

- 📁 **Stick to the folder structure** — `CSE-XX/Subject/YourFile`
- 🏷️ **Name files clearly** — e.g., `Lab-03-Greedy-Algorithm.pdf`
- 📝 **Add a short description** in your PR
- ✅ **No copyrighted material** — only your own work or properly credited resources
- 🤗 **Be respectful** — we're all learners here!
